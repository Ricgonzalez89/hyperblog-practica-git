# hyperblog-practica-git 💚
Un repositorio de practica para el manejo de Git y un pequeño blog desarrollado a lo largo del curso como parte de las prácticas.
>En este curso he podido estudiar más a fondo el uso de los comandos básicos de Git, la increíble herramienta que es Git Bash para la gestión de cambios y versiones de un proyecto, así como conceptos fundamentales y buenas prácticas para el trabajo con esta herramienta.
> - Sin duda, recomiendo el curso de Git de Platzi.

## En este curso se aprende Git desde sus bases

* Comandos basicos y avanzados de Git.
* Buenas prácricas y recomendaciones.
* Trucos loquísimos que esconde Git bajo la manga.
